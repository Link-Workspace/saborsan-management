# Saborsan Gestão - Sistema Demonstrativo

Sistema de gestão demonstrativo da Saborsan, feito em **Vite + React**, com dados hardcoded e interface responsiva para desktop e mobile.

## Como rodar no VS Code

```bash
npm install
npm run dev
```

Depois acesse o endereço mostrado no terminal, normalmente:

```bash
http://localhost:5173
```

## Acesso demo

Você pode entrar com qualquer e-mail e senha. Sugestão:

- E-mail: `admin@saborsan.com`
- Senha: `123456`

## Funcionalidades demo incluídas

- Tela de login para funcionário
- Dashboard de gestão
- Recebimento de pedidos vindos do app
- Detalhes e status dos pedidos
- Geração demonstrativa de nota fiscal
- Gestão de estoque
- Fornecedores e compras
- Comunicação com fornecedores
- Entregas e rotas
- Clientes
- Financeiro
- Relatórios
- Central de automação/IA com opção de ligar/desligar
- Layout responsivo para celular

Tudo é hardcoded para demonstração visual e funcional.
